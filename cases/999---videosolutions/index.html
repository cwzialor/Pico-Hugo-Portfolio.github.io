Video Solutions HTML & CSS Website

VideoSolutions is a Japanese social media management firm that provides international clients with localised content and campaigns in the luxury industry. They require a website to help inform potential clients of the products and services they have available.

The owner built the original site in Tumult Hype, that is not the intended purpose of the software. As a consequence the site was slow and buggy. I was tasked with rebuilding the site using HTML and CSS.

Following the original design, HTML and CSS allowed to recreate the site that uses less memory, server space and processing power to access.

The outcome was a minimal mobile an site with functional meta data for Search Engine Optimisation.
